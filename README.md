# MyFXJournal

This public repository is a lightweight overview for MyFXJournal.

The production application code is intentionally private. That includes:

- MT5 setup, sync, and account-linking workflows
- import/parsing logic and deduplication rules
- analytics and review-generation logic
- background workers, internal routes, and operational tooling
- database schema, migrations, and test coverage

This repo is kept minimal on purpose so the public project page can exist without exposing product moat or internal implementation details.
